# Python Automation Scripts

This repository contains some Python automation scripts designed to automate repetitive tasks and streamline workflows.

## Purpose

The goal of this repository is to build a collection of useful automation tools.



## Getting Started

### Prerequisites

- Python 3.x installed
- Required packages (install as needed for each script)

### Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/RumeshChathuranga/Python-Automation.git
   cd Python-Automation
   ```

2. Install required packages for specific scripts
